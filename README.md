### Como fazer ele
Voçes vão ver que se tem branchs separadas para cada issue e cada um so vai fazer o commit nessa branchs e depois pedir o pull request, pois ai vou juntar aceitar e vai ficar tudo na main.

Voçes pode fazer o commite do jeito que quiser copiando e colando ou como eu faço usando o terminal e chave ssh.

Mas ainda assim tem que ter o git intalado no pc para fazer a sincronização com a main e fazer o clone do repositorio.
Para fazer a clonagem pode ser de sua maneiras:
* Com chave ssh:
  ```bash
  git clone git@github.com:Cassiano-C/Topicos_II.git
  ```
* Sem chave ssh:
  ```bash
  git clone https://github.com/Cassiano-C/Topicos_II.git
  ```
E para se fazer o sincronismo é assim:
```bash
git pull origin main 
```
